# Task Completion Checklist
Before claiming a task is DONE:
1. **Lint**: Run `ruff check .`
2. **Test**: Run `pytest` and specific feature tests.
3. **Verify**: Evidence before assertions (e.g., run verification scripts).
4. **Document**: Update `task.md` and `walkthrough.md`.
5. **Memory**: Update Serena memories if architecture/patterns changed.
6. **Clean**: Remove temp logs/artifacts.
